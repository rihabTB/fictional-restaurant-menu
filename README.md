# 🍽️ Dar Khedawj Menu

A simple restaurant menu website showcasing traditional Algerian dishes.

## ✨ Features

* Display categorized menu (Appetizers, Main Courses, Desserts, Drinks)
* Clean card layout for each dish
* Smooth navigation with navbar
* Simple and responsive design

## 🛠️ Tech

* HTML
* CSS

## 🚀 Run

Just open `index.html` in your browser.

## 📌 Note

This project was built for practicing layout design and basic frontend structure.
